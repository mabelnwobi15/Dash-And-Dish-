Fast Food Delivery at Your Fingertips

Dash & Dish is a mobile food delivery application built using React Native and Expo. The app allows users to browse meals, add items to their cart, and complete orders through a clean and intuitive interface.

📌 Project Overview

This project was developed as a collaborative group project through the Code to Career community.

The main goal of this project was to:

Practice team collaboration in a development environment
Improve communication and coordination skills
Build a real-world mobile application using modern tools

Dash & Dish simulates a food delivery experience, focusing on usability, design, and smooth navigation.

🎯 Key Features
🍔 Browse meals from a structured menu
📄 View detailed dish information
🛒 Add items to a cart
💳 Complete a checkout process
✅ Receive order confirmation with delivery time
📜 View order history
🔐 Login interface
🧭 Bottom tab navigation (Home, Cart, Menu, Profile)

Framework: React Native (Expo)
Routing: Expo Router
Language: JavaScript / TypeScript
Testing: Expo Go & web browser

How to run the app: 😎
git clone <your-repo-link>
cd DashAndDish
npm install
npx expo start
